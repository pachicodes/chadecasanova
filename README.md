# Cha de Casa Nova

Site simples para lista de presentes de cha de casa nova.

## Funcionalidades

- Lista de presentes com status de disponibilidade.
- Botao para marcar item como comprado.
- Pop-up de confirmacao antes de finalizar a compra.
- Item marcado aparece riscado e com status "Comprado".

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Clique em "Marcar como comprado" no item desejado.
3. Confirme no pop-up para finalizar.

## Estrutura

- `index.html`: estrutura da pagina.
- `styles.css`: estilos visuais e responsividade.
- `script.js`: logica da lista e confirmacao.
